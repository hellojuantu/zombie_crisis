# 僵尸危机

一个轻量的多人联机恐怖撤离生存游戏。服务端权威模拟玩家、感染体、子弹、任务物、撤离终端、背包、整备和关卡推进；前端负责输入预测、插值、HUD、音效和 Canvas 渲染。

## 运行

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python3 server_asgi.py
```

打开：

```text
http://localhost:8080/
```

局域网联机时，让朋友访问你的局域网 IP：

```text
http://你的局域网IP:8080/
```

## 操作

- WASD：移动
- 鼠标：瞄准
- 左键：开火，怪物贴脸时自动切刀近战
- 右键 / Q / E：切换已解锁武器
- R：换弹
- 1-5：切换已解锁武器
- 空格 / Shift：冲刺
- 背包图标：打开全屏背包，查看武器、物资、撤离条件和路线奖励

## 玩法循环

- 关卡推进：每一关生成不同设施楼层，完成撤离条件并读条成功后进入整备，由玩家主动进入下一关
- 收集任务：保险丝主要在设施里找，病毒样本和门禁卡需要击杀特定感染体掉落
- 多撤离点：每关有多个撤离点，地图和小地图会标记 1/2/3，不同撤离点要求和奖励不同
- 恐怖探索：撤离点需要靠近或通过机房供电后显形，HUD 会显示缺什么和哪个出口可撤
- 设施探险：病房可止血但会引怪；机房消耗保险丝并定位撤离点；样本库提高样本掉落但触发共振；仓库能搜出武器、载具和补给，但会触发警报
- 武器解锁：手枪、步枪、散弹枪、冲锋枪、爆破枪拥有不同弹匣、射速、散布、穿透和爆炸效果
- 载具窗口：维修推车提供限时高速推进，可撞开近身感染体，但会把玩家推向更高风险路线选择
- 游戏导演：低频检查战场热度，把远离玩家的僵尸重新拉回压力圈，减少空场断档
- 僵尸避障：僵尸遇到设施墙体会选边绕行，不再只会直线撞墙
- 连杀奖励：连杀 10 获得短速射，连杀 20 获得短三连发，连杀 30 获得短护盾
- 精英压力：波次会逐步解锁爬行者、重型、毒性、装甲、跳扑、尖啸、爆裂体
- 特殊行为：跳扑怪会短距离爆发，尖啸怪会鼓舞附近尸群，爆裂体死亡后会范围爆炸
- 迷雾事件：回血、搜仓库、样本房共振和撤离读条会触发不同场景的雾潮压力，主波次打完也可能有有限感染源余压
- Boss 节点：第 3/6/9... 关出现巨型感染体，形成阶段高潮和高价值掉落来源
- 主线结局：第 6 关完成撤离会揭露 B13 层真相，之后可继续进入无尽层挑战
- 叙事钩子：每层有故事广播，撤离和档案碎片会继续揭露设施真相
- 声音反馈：持续恐怖底噪、危险压迫音、僵尸死亡惊吓音和关键事件音效构成节奏反馈
- 全服榜：服务端下发 top 8，前端不再用视野内玩家冒充全服排行

## 为什么会让玩家继续玩

- 短循环：移动、开枪、贴脸自动出刀、击杀掉落、拾取反馈在几秒内完成，保证每次操作都有回应。
- 中循环：每层的撤离条件不同，玩家会在“缺物资、找设施、打特殊怪、冒险撤离”之间做选择。
- 长循环：武器解锁、路线奖励、档案碎片、Boss 节点和逐层变化的设施布局，给玩家继续下一关的理由。
- 风险回报：病房回血会引怪，样本库提高掉落会施压，仓库给好东西但可能刷压力包，载具快但容易冲进危险区。
- 社交驱动：多人同屏、全服榜、路线选择和互相救场会制造“再来一把”的讨论点。

目前仍建议继续通过真实局域网多人试玩收集数据：首局教学完成率、第一关撤离率、第三关留存率、平均 Ping、死亡后是否继续加入。这些数据能判断“上瘾感”是否来自真实玩家行为，而不只是设计判断。

## 架构

- `server_game/`：服务端权威模拟，不包含浏览器或渲染逻辑
- `server_asgi.py`：ASGI + Socket.IO 生产入口
- `static/js/game/`：前端游戏主循环、渲染、HUD、特效
- `static/js/protocol.js`：紧凑快照协议解码
- `tools/load_test.py`：Socket.IO 压测脚本

## 压测

```bash
python3 tools/load_test.py --clients 100 --duration 10 --ramp 0.01 --input-interval 0.066 --shoot-duty 0.85
```

当前优化重点：

- WebSocket only，避免轮询退化
- 服务端权威，客户端预测与插值
- AOI 视野裁剪，减少 100 人同步包大小
- 网络快照 16Hz，服务端模拟 30Hz，兼顾局域网多人压力和移动平滑度
- 高频事件按附近玩家定向发送
- 玩家 HUD 只保留生命、武器、弹药、目标和轻量 Ping，调试指标不常驻
